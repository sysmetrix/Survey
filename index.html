📋 분석 절차 안내
1
데이터 준비Excel 또는 CSV 설문 결과 파일 준비. 1행 변수명, 2행부터 응답 데이터.
2
파일 업로드위 영역에 파일 드래그 또는 클릭. 교차변수·척도변수 자동 감지.
3
교차변수 선택좌측 사이드바에서 분석 기준(성별, 연령대, 지역 등) 선택.
4
분석 결과 확인6개 탭에서 시각화와 자동 분석 리포트 확인.
5
보고서 추출DOCX 또는 Google Docs 형식으로 전체 분석 결과를 문서로 내보내기.
📌 데이터 형식 가이드
교차변수 — 성별, 연령대, 지역, 신분 등 (2~20개 범주)
척도변수 — 1~5점 리커트 척도 (정수 값)
변수 유형은 자동 감지됩니다.
