📋 분석 절차 안내
1
데이터 준비Excel(.xlsx) 또는 CSV. 1행 = 변수명, 2행부터 응답. 아래 템플릿 버튼으로 예시 파일 다운로드 가능.
2
파일 업로드드래그 또는 클릭. 교차변수·척도변수·주관식 자동 감지. EUC-KR·UTF-8 인코딩 자동 처리.
3
교차변수 선택사이드바에서 성별·연령대 등 클릭 → 교차분석 모드 자동 전환. 여러 교차변수 순서대로 선택 가능.
4
모드 전환 · 시각화상단 모드바에서 종합분석 → 교차분석 → 심층분석 전환. 서브탭으로 히트맵·점플롯·분포 등 개별 선택.
5
심층 통계 검증심층분석 탭 → t-test·ANOVA·Tukey HSD 사후검정·상관행렬·회귀분석·카이제곱. 모든 결과에 자동 해설 포함.
6
보고서 내보내기헤더의 📄 PDF · 📥 DOCX 버튼으로 자동 보고서. 📋 리포트 복사로 GDocs·Word 붙여넣기. ⚙️ 설정에서 기관명·제목 변경.
📌 데이터 형식 가이드
교차변수 — 성별·연령대·지역·신분·소속 등 범주형 (2~20개 값, 자동 감지)
척도변수 — 1~5점 리커트 척도 정수값 (고유값 ≤ 10인 숫자 컬럼 자동 감지)

· EUC-KR/UTF-8 자동 처리 · 권장 최대 5MB · 불성실 응답(straight-lining) 자동 탐지
❓ 자주 묻는 질문
Q. 분석 결과가 외부 서버에 저장되나요?
아니요. 모든 데이터 처리는 브라우저 내에서만 이루어지며 어떠한 서버에도 전송되지 않습니다. 개인정보를 포함한 설문 데이터를 안전하게 사용할 수 있습니다.
Q. 척도변수가 자동 감지되지 않아요.
척도변수 컬럼의 값이 1~5 사이 정수이어야 합니다. 빈 셀이 많거나 값이 문자(①②③)인 경우 감지되지 않습니다. 숫자 1~5로 변환 후 재업로드하세요.
Q. 분석 이력이 사라졌어요.
이력은 해당 브라우저의 localStorage에 24시간만 보존됩니다. 중요한 분석은 📥 DOCX 또는 📄 PDF로 저장하거나, 이력 화면에서 JSON으로 내보내기하세요.
Q. 집단 수가 너무 많아 히트맵이 잘려요.
교차변수의 범주가 15개 이상이면 열이 좁아집니다. 범주를 통합하거나 상세표 탭을 이용하세요.
